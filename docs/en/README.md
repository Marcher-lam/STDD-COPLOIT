## Contributing Translations

To add or improve translations:

1. Create/edit files in `docs/en/` for English
2. Add language cross-reference links at the bottom of each file
3. Keep translations in sync with Chinese originals
4. Technical terms (commands, config keys) should NOT be translated

### Translation Guidelines

| Keep Original | Translate |
|---------------|-----------|
| `/stdd:propose` | All prose descriptions |
| `stdd/config.yaml` | Usage instructions |
| `Ralph Loop` | Feature explanations |
| `BDD`, `GWT` | Conceptual descriptions |
| `Constitution` | User-facing documentation |
